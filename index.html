Setting Up a hugo static site on GitHub pages

29 May, 2016

Instructions for setting up a static site on github pages using hugo. Set up a hugo site in a new directory md ~/blog cd ~/blog hugo new site . Clone the theme directory inside the ~/blog directory git clone https://github.com/enten/hyde-y.git themes/hyde-y themes/hyde-y Create the ~/blog/data folder and create a file called Menu.toml with the following [posts] Name = "posts" Title = "Show list of posts" URL = "/post" [topics] Name = "topics" Title = "Show list of topics" URL = "/topics" [tags] Name = "tags" Title = "Show list of tags" URL = "/tags" Create a new post, still in the ~/blog directory hugo new post/setting-up-hugo-github-pages.md Edit the config.toml file in the ~/blog directory baseurl = "/" languageCode = "en-us" title = "Lowsparkk Labs" theme = "hyde-y" [permalinks] post = "/:year/:month/:day/:slug/" code = "/:slug/" [taxonomies] tag = "tags" topic = "topics" [author] name = "Low Sparkk" email = "low.sparkk@yahoo.com" [params] # You can use markdown here.
